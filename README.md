# Charcoals React + TypeScript + Vite

هذا المشروع الآن يعمل بـ React وTypeScript وVite فقط.

## الأوامر

- تشغيل التطوير:
```
npm run dev
```
- بناء المشروع:
```
npm run build
```
- معاينة البناء:
```
npm run preview
```

## المتطلبات
- node.js
- npm

## ملاحظات
- تم حذف جميع إعدادات وملفات Vue.
- إذا أردت استخدام Tailwind CSS أضف ملف tailwind.config.js وركب الحزمة.
